# 学科适配策略

> 本文件为 `exam-paper-error-analysis` 技能的参考数据，提供职业教育课程的学科适配指南。

## 职业教育课程适配

当学科为职业教育课程时，按以下维度适配分析框架：

### 1. 机电类（数控技术、模具设计、电气自动化等）

**核心能力维度**：
- 图纸识读能力（三视图、剖视图、装配图）
- 工艺规程编制能力
- 设备操作与编程能力（G代码、M代码、PLC编程）
- 质量检测与误差分析能力
- 安全生产规范意识

**典型错误场景**：
- 不会读三视图，无法从二维图纸想象三维结构
- 工艺路线安排不合理（先粗后精、先面后孔等原则）
- 编程指令使用错误（G00/G01/G02/G03混淆）
- 量具使用不规范（游标卡尺、千分尺读数错误）
- 忽视安全操作规程（未戴防护眼镜、未锁紧工件等）

**分析要点**：
- 区分"理论不会"和"操作不规范"
- 关注国家职业标准对应能力项（如车工中级工标准）
- 结合学校实训设备条件给出可执行建议

### 2. 计算机类（软件开发、网络工程、信息安全等）

**核心能力维度**：
- 编程逻辑与算法设计能力
- 调试与排错能力
- 系统分析与设计能力
- 文档编写与规范意识
- 团队协作与项目管理能力

**典型错误场景**：
- 循环边界条件处理错误（off-by-one error）
- 异常处理缺失（未捕获空指针、数组越界等）
- 数据库设计不规范（未遵循第三范式）
- 代码注释不完整，逻辑难以理解
- 版本控制使用不当（未写commit message、未分支管理）

**分析要点**：
- 关注代码可执行性（不仅是理论正确）
- 强调工程规范（命名规范、注释规范、测试覆盖）
- 结合真实项目场景分析

### 3. 会计类（财务会计、成本会计、审计等）

**核心能力维度**：
- 会计准则应用能力
- 账务处理能力
- 财务报表编制能力
- 税务计算与申报能力
- 内部控制与风险管理能力

**典型错误场景**：
- 借贷方向记反（应收账款 vs 预收账款）
- 会计科目使用错误（管理费用 vs 销售费用）
- 折旧计算错误（年限平均法 vs 双倍余额递减法）
- 税务处理错误（增值税进项税额抵扣条件）
- 财务报表勾稽关系错误（资产负债表不平）

**分析要点**：
- 强调会计准则的准确应用
- 关注实务操作中的常见误区
- 结合最新税法政策分析

### 4. 护理类（基础护理、内科护理、外科护理等）

**核心能力维度**：
- 护理评估能力
- 护理计划制定能力
- 护理操作技能
- 病情观察与应急处理能力
- 健康教育与沟通能力

**典型错误场景**：
- 护理诊断不准确（护理问题 vs 医疗诊断混淆）
- 给药剂量计算错误（药物浓度换算错误）
- 操作顺序错误（无菌操作原则违反）
- 病情观察遗漏（生命体征变化未及时发现）
- 健康教育不到位（患者依从性差）

**分析要点**：
- 强调患者安全红线（给药安全、无菌操作）
- 关注护理程序的完整性（评估-诊断-计划-实施-评价）
- 结合临床真实案例分析

### 5. 汽修类（汽车检测与维修、新能源汽车技术等）

**核心能力维度**：
- 故障诊断与分析能力
- 检测设备使用能力
- 维修工艺执行能力
- 安全操作规范意识
- 客户服务与沟通能力

**典型错误场景**：
- 故障码解读错误（P0300 vs P0301含义混淆）
- 检测工具使用不当（万用表量程选择错误）
- 维修工艺不规范（扭矩未按标准拧紧）
- 安全防护不到位（未断开蓄电池负极）
- 维修记录不完整（故障现象、处理过程未记录）

**分析要点**：
- 强调故障诊断的逻辑思维（从现象到本质）
- 关注安全操作规范（高压电安全、举升机安全）
- 结合真实故障案例分析

### 6. 建筑类（建筑工程技术、工程造价、建筑设计等）

**核心能力维度**：
- 图纸识读与绘制能力
- 工程量计算能力
- 施工组织设计能力
- 质量检验与验收能力
- 安全生产管理能力

**典型错误场景**：
- 图纸尺寸标注错误（比例尺使用不当）
- 工程量计算错误（未扣除门窗洞口面积）
- 施工方案不合理（未考虑季节性施工措施）
- 质量验收标准不熟悉（主控项目 vs 一般项目）
- 安全隐患识别不足（脚手架搭设不规范）

**分析要点**：
- 强调国家建筑规范标准的准确应用
- 关注施工安全红线
- 结合真实工程案例

### 7. 旅游类（酒店管理、导游服务、旅游管理等）

**核心能力维度**：
- 服务流程执行能力
- 客户沟通能力
- 应急处理能力
- 产品设计与营销能力
- 跨文化交际能力

**典型错误场景**：
- 服务流程遗漏（未进行入住登记）
- 客户沟通不当（未使用礼貌用语）
- 应急处理不及时（客人投诉未及时处理）
- 产品设计不合理（未考虑目标客群需求）
- 跨文化误解（未尊重客人宗教信仰）

**分析要点**：
- 强调服务意识和客户体验
- 关注行业标准和服务规范
- 结合真实服务场景

### 8. 电商类（电子商务、网络营销、跨境电商等）

**核心能力维度**：
- 平台运营能力
- 数据分析能力
- 营销策划能力
- 客户服务能力
- 供应链管理能力

**典型错误场景**：
- 平台规则不熟悉（违规操作导致店铺扣分）
- 数据分析错误（转化率计算错误）
- 营销活动策划不合理（未考虑目标受众）
- 客户服务不到位（未及时回复客户咨询）
- 供应链管理混乱（库存积压或缺货）

**分析要点**：
- 强调数据驱动的决策思维
- 关注平台规则和合规要求
- 结合真实电商案例

## 通用职教分析原则

1. **对接岗位标准**：分析要关联国家职业标准或职业技能等级证书要求
2. **区分理论与实践**：明确"理论不会做"和"操作不规范"属于不同维度
3. **关注安全红线**：涉及安全规程的错误必须标注为"⚠️ 高风险"
4. **考虑实训条件**：建议需贴合学校实际实训设备条件
5. **强调"做中学"**：讲评和练习建议以实操训练为主
6. **真实工作情境**：变式题设计要更换工作情境/设备型号
