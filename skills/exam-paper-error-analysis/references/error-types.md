## 1. 知识不会型

### 描述
学生对某个知识点完全没有掌握或掌握不完整，导致无法正确作答。这不是"粗心"或"看错"，而是根本不知道怎么做。

### 典型表现
- 空白不答或随意填写
- 使用了完全错误的方法或公式
- 对核心概念表述错误

### 学科示例

| 学科 | 示例 |
|------|------|
| 数学 | 分数应用题做错，因为没有理解"单位1"的含义 |
| 语文 | 阅读题答偏，因为没有掌握人物描写方法（外貌、语言、动作、心理） |
| 英语 | 时态填空全错，因为不知道现在完成时的构成和用法 |
| 物理 | 电路题不会做，因为不知道欧姆定律的适用条件 |
| 机电 | 不会读三视图，因为没有掌握投影规律 |
| 护理 | 给药剂量计算错误，因为不知道药物浓度换算公式 |

### 讲评策略
1. **补关键知识**：不讲整章，只补这道题缺的那个知识点
2. **最小化讲解**：用 3-5 分钟讲清楚核心概念
3. **即时巩固**：讲完立刻做 1-2 道同类题检验
4. **避免**：不要从头到尾重讲一遍，针对性要强

## 2. 概念混淆型

### 描述
学生背过定义、能说出公式，但内心理解是错的。把两个相似但不同的概念混为一谈，或者对概念的理解存在偏差。

### 典型表现
- 能说出定义，但应用时张冠李戴
- 用错误的概念解释现象
- 在辨析题中选错

### 学科示例

| 学科 | 示例 |
|------|------|
| 数学 | 知道"平均数"，却把平均数当成真实存在的某一个数 |
| 语文 | 以为只要句子里有"像"就是比喻句（忽略了本体和喻体必须是不同类事物） |
| 英语 | 混淆 affect（动词）和 effect（名词） |
| 物理 | 混淆"速度"和"加速度"，认为速度大加速度就大 |
| 化学 | 混淆"物理变化"和"化学变化"的判断标准 |
| 会计 | 混淆"应收账款"和"预收账款"的借贷方向 |

### 讲评策略
1. **对比辨析**：把两个易混淆概念放在一起对比
2. **找本质区别**：不是背定义，而是理解本质差异
3. **举反例**：用反例帮助学生区分
4. **设计辨析题**：让学生判断哪个概念适用

## 3. 审题粗心型

### 描述
学生不是不会做，而是没读懂题目要求。漏看条件、看错关键词、没有区分"求什么"和"已知什么"。

### 典型表现
- 答案方向完全错误（求最大值答了最小值）
- 漏掉关键条件导致结果错误
- 单位换算遗漏

### 学科示例

| 学科 | 示例 |
|------|------|
| 数学 | 题目问"至少需要多少米"，答了"最多" |
| 语文 | 题目问"人物特点"，答了"人物感受" |
| 英语 | 题目要求"用过去时"，用了现在时 |
| 物理 | 题目给的是"直径"，当成"半径"代入公式 |
| 汽修 | 题目要求"拆卸顺序"，答了"装配顺序" |
| 护理 | 题目要求"静脉注射"，按"肌肉注射"操作 |

### 讲评策略
1. **训练读题方法**：圈画关键词、划出已知和所求
2. **转述题意**：让学生用自己的话复述题目要求
3. **对比训练**：给两道相似题，只有一个关键词不同，让学生找差异
4. **避免**：不要只说"下次认真点"，要给具体方法

## 4. 方法断裂型

### 描述
学生知道答案方向，但解题过程断了。会列式但不知道为什么这样列，会判断但说不出理由，会写结果但中间缺少推理。

### 典型表现
- 能写出第一步，但第二步就断了
- 答案对但过程不完整
- 能模仿例题，但说不清每一步的理由

### 学科示例

| 学科 | 示例 |
|------|------|
| 数学 | 会列方程但不会解方程，或解到一半不会变形 |
| 语文 | 能找出关键句，但不会分析其表达效果 |
| 物理 | 知道用能量守恒，但不会列能量守恒方程 |
| 计算机 | 知道要用循环，但循环条件写错或边界处理缺失 |
| 建筑 | 知道要算受力，但受力分析图不完整 |

### 讲评策略
1. **补过程支架**：把解题过程拆成小步骤，每步给提示
2. **追问"为什么"**：让学生说出每一步的理由
3. **思维链条训练**：用"因为…所以…"的句式串联推理过程
4. **半填空练习**：给出不完整的解题过程，让学生补中间步骤

## 5. 表达不完整型

### 描述
学生心里明白，但表达不出来或表达不完整。缺少关键步骤、缺少结论、语言组织混乱。

### 典型表现
- 答案只写了一半
- 缺少必要的推理过程
- 语言表述不准确、不完整

### 学科示例

| 学科 | 示例 |
|------|------|
| 语文 | 阅读理解只写了结论，没有结合文本分析 |
| 物理 | 实验题只写了结论，没有写实验步骤和现象 |
| 化学 | 方程式配平了，但没有写反应条件 |
| 旅游 | 服务流程描述了，但缺少应急处理环节 |
| 电商 | 营销策划写了方案，但没有数据支撑 |

### 讲评策略
1. **给答题句式**：提供标准答题模板（如"观点+论据+分析"）
2. **分步表达训练**：先写结论，再补理由，最后加证据
3. **对照评分标准**：让学生对照评分标准检查自己的答案
4. **互评训练**：同学互相检查答案是否完整

## 6. 迁移失败型

### 描述
学生在课堂例题里会做，到了作业或考试就错。说明可能只是记住了题型，而没有理解方法。一换情境、一换材料、一换问法，原来的方法就不会用了。

### 典型表现
- 课堂例题能做对，作业题就错
- 换一种问法就不会了
- 换一个实际情境就不知道怎么应用

### 学科示例

| 学科 | 示例 |
|------|------|
| 数学 | 课堂例题是"甲乙相向而行"会做，换成"两车同向追及"就不会 |
| 语文 | 课内阅读分析会做，课外阅读就不会 |
| 物理 | 理想条件下的计算会做，加入摩擦力就不会 |
| 机电 | 标准工件加工会做，换非标工件就不会 |
| 护理 | 标准病例分析会做，遇到并发症就不会 |
| 汽修 | 常见故障会诊断，遇到复合故障就不会 |

### 讲评策略
1. **变式练习**：换数字、换情境、换问法、增加干扰条件
2. **方法提炼**：从具体题目中抽象出通用方法
3. **对比分析**：让学生比较例题和变式题的异同
4. **真实情境**：把题目放到真实工作/生活情境中

## 7. 跨学科错误类型映射

同一个错误类型在不同学科中的具体表现：

### 知识不会型
- 理科：公式不会用、概念不知道
- 文科：知识点没背过、方法不知道
- 职教：操作步骤不知道、设备不会用

### 概念混淆型
- 理科：相似公式混用、物理量混淆
- 文科：修辞手法混淆、语法概念混淆
- 职教：工艺方法混淆、设备功能混淆

### 审题粗心型
- 理科：看错数字、漏看条件、单位遗漏
- 文科：看错问题要求、漏看限定词
- 职教：看错参数要求、漏看安全提示

### 方法断裂型
- 理科：解题过程断在某一步
- 文科：分析过程不完整
- 职教：操作流程断在某一步

### 表达不完整型
- 理科：缺少计算过程或单位
- 文科：缺少文本证据或分析
- 职教：缺少操作记录或安全确认

### 迁移失败型
- 理科：换题型就不会
- 文科：换文本就不会
- 职教：换设备/换情境就不会
