# Exam Paper Error Analysis Skill - Design Spec

## Overview

全学科错题分析技能，覆盖基础教育和职业教育（机电、计算机、会计、护理、汽修、建筑、旅游、电商等）8+专业。

## Core Capabilities

1. **单题深度分析** - 七步框架分析单道错题
2. **整卷失分分析** - 失分地图 + 提分优先级
3. **变式题生成** - 基础巩固 + 易错辨析 + 迁移应用
4. **错题分类讲评** - 六类错误分类 + 讲评策略

## Architecture

```
skills/exam-paper-error-analysis/
├── SKILL.md                    # 主技能文件（触发规则、输入校验、分析原则、子能力路由、输出规范）
└── references/
    ├── error-types.md          # 六类错误定义（知识不会型、概念混淆型、审题粗心型、方法断裂型、表达不完整型、迁移失败型）
    ├── html-template.md        # HTML报告结构定义 + 模板代码
    ├── subject-adaptation.md   # 学科适配策略（职业教育8大专业）
    └── vocational-standards.md # 岗位能力模型（国家职业标准、职业技能等级证书）
```

## Key Design Decisions

1. **多模态识别**：强制使用 agent 内置多模态能力，禁止本地 OCR
2. **PDF 处理**：统一 pipeline - PDF → pypdfium2 转图片 (scale=2) → 多模态识别
3. **HTML 输出**：单一完整文件，包含全部 6 个章节结构
4. **条件加载**：参考文件按 MANDATORY/推荐 指示加载，不预加载
5. **自我校验**：分析结果与参考答案一致性检查

## Error Type Classification

| 类型 | 描述 | 讲评策略 |
|------|------|----------|
| 知识不会型 | 完全不知道怎么做 | 补关键知识 |
| 概念混淆型 | 理解有偏差 | 对比辨析 |
| 审题粗心型 | 没读懂题目要求 | 训练读题方法 |
| 方法断裂型 | 解题过程断了 | 补过程支架 |
| 表达不完整型 | 心里明白但说不出来 | 给答题句式 |
| 迁移失败型 | 换情境就不会 | 做变式练习 |

## Vocational Education Support

8 大专业：机电、计算机、会计、护理、汽修、建筑、旅游、电商

每个专业包含：
- 核心能力维度
- 典型错误场景
- 国家职业标准对应能力项
- 职业技能等级证书要求

测试覆盖：
